# Exotics: Reptil medisin - Diagnostikk

## Norsk
Denne undergruppen er opprettet for å gi plass til diagnostiske reptildokumenter i samme hovedstruktur som miljøkrav og behandling.

- I den nåværende dokumentpakken fantes det ikke egne PDF-er for diagnostikk.
- Kategorien er derfor opprettet som strukturplassholder for senere innhold.
- Miljø- og biologiinformasjon er lagt under `Miljøkrav og biologi`.

## English
This subgroup is created so diagnostic reptile documents can live in the same main structure as husbandry and treatment content.

- The current source bundle did not include dedicated diagnostic PDFs.
- This category is therefore created as a structural placeholder for later content.
- Husbandry and biology information has been placed under `Miljøkrav og biologi`.
